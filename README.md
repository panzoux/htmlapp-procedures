# Procedure Manager

## 概要

Procedure Managerは、作業手順をCSV形式で管理できるシングルページHTMLアプリケーションです。作業の準備フェーズと実行フェーズを明確に分離し、手順の順次実行を強制することで、作業ミスを防ぎます。

## 主な機能

### 🎯 2つのモード
- **PREPAREモード（準備モード）**: 手順の作成・編集が可能
  - 手順内容の編集
  - ステップの追加・削除
  - ステータス変更は無効化
- **EXECUTEモード（実行モード）**: 手順の実行が可能
  - ステータスの更新
  - 備考欄の記入
  - 手順内容の編集は無効化

### 📋 順次実行ロジック
EXECUTEモードでは、手順の順次実行が強制されます：
- 最初のステップは常に開始可能
- 2番目以降のステップは、**前のステップが「Pending」状態でない場合のみ**開始可能
- つまり、前のステップが「In Progress」または「Done」になっていれば、次のステップを開始できます
- この仕組みにより、手順の飛ばしや順序の間違いを防止します

### 📊 ステータス管理
各ステップは3つのステータスを持ちます：
- **Pending（待機中）**: 未着手の状態
- **In Progress（進行中）**: 作業中の状態（開始時刻が自動記録）
- **Done（完了）**: 完了した状態（終了時刻が自動記録）

ステータスボタンをクリックすると、Pending → In Progress → Done → Pending の順に循環します。Pendingに戻すと、開始時刻と終了時刻の両方がクリアされます。

### 💾 CSV インポート/エクスポート
- **UTF-8対応**: 日本語を含むテキストを正しく扱えます
- **自動ID採番**: インポート時にIDが自動的に振り直されます
- **Excel互換**: BOM付きUTF-8でエクスポートされ、Excelで直接開けます
- **改行対応**: 手順や備考欄に改行が含まれていても正しく処理されます

### 🎨 モダンなUI
- ダークモード対応
- グラスモーフィズムデザイン
- スムーズなアニメーション
- 進行中のタスクにはパルスエフェクト

## 使い方

### 1. アプリケーションの起動
`index.html` をダブルクリックするか、ブラウザにドラッグ&ドロップして開きます。

### 2. 手順の準備（PREPAREモード）
1. 「+ ADD STEP」ボタンで新しいステップを追加
2. 「Procedure」列をクリックして手順内容を入力
3. 不要なステップは右端の「×」ボタンで削除
4. 「EXPORT CSV」で作業手順をCSVファイルとして保存

### 3. 手順の実行（EXECUTEモード）
1. ヘッダーの「EXECUTE」ボタンでモードを切り替え
2. 最初のステップのステータスボタンをクリックして開始
3. 作業が完了したら再度ステータスボタンをクリックして「Done」に変更
4. 「Remarks」列に作業メモや気づきを記入
5. 前のステップが完了または進行中になると、次のステップが開始可能になります

### 4. CSVのインポート
1. 「IMPORT CSV」ボタンをクリック
2. 以前エクスポートしたCSVファイルを選択
3. 手順とステータスが復元されます

## CSVフォーマット

```csv
ID,Procedure,Status,Start Time,End Time,Remarks
1,"サーバーにログイン",Pending,,,
2,"バックアップを取得",Pending,,,
3,"アプリケーションを停止",Pending,,,
```

- 1行目: ヘッダー行（インポート時は自動判定）
- 各列: ID, 手順, ステータス, 開始時刻, 終了時刻, 備考
- 改行を含む場合はダブルクォートで囲む
- ダブルクォート自体は `""` でエスケープ
